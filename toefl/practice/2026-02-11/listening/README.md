## üéß Practice Instructions

### For Each Recording:

1. **Prepare your materials:**
   - Blank paper for Cornell-style notes
   - Timer
   - Quiet environment

2. **Listen and take notes:**
   - Play the MP3 file ONCE
   - Take notes while listening
   - Do NOT read the script file

3. **Answer questions:**
   - Open the questions file (2-*-questions.md)
   - Answer based on your notes and memory
   - Do NOT replay the audio

4. **Check your answers:**
   - Open the answers file (3-*-answers.md)
   - Review explanations for all questions
   - Analyze your performance

5. **Review and learn:**
   - Read the transcript while listening again
   - Note vocabulary and phrases you missed
   - Identify patterns in your errors

- 14-16: Strong (TOEFL 24-27 range)
- 11-13: Good (TOEFL 20-23 range)

- Below 8: Needs improvement (TOEFL below 16)

## üìö Study Tips

**For Conversations:**
- Focus on the purpose of the interaction
- Track multiple solutions or options discussed
- Note specific details (dates, numbers, requirements)
- Identify the speakers' attitudes and concerns

**For Lectures:**
- Identify the main topic and organizational pattern
- Note definitions of technical terms
- Track cause-effect relationships
- Distinguish main ideas from supporting examples
- Pay attention to professor's emphasis and repetition
